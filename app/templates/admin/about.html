Tentang SentimentApp

Informasi tentang aplikasi dan metodologi yang digunakan.

Apa itu SentimentApp?

SentimentApp adalah aplikasi web untuk menganalisis sentimen ulasan pengguna aplikasi ChatGPT dan Gemini AI. Dengan menggunakan algoritma Random Forest dan teknik Natural Language Processing (NLP), aplikasi ini mengklasifikasikan ulasan menjadi sentimen Positif, Negatif, atau Netral secara otomatis.

Fitur Utama
  • Analisis Sentimen — Klasifikasi otomatis menggunakan Random Forest
  • Riwayat Lengkap — Simpan dan lihat seluruh riwayat analisis
  • Kelola Dataset — Impor, preprocess, dan latih model dari CSV
  • Evaluasi Model — Confusion Matrix, Precision, Recall, F1-Score
  • Aman & Privat — Akun pribadi untuk setiap pengguna
Metodologi NLP
Preprocessing
Case Folding Cleaning Tokenisasi Normalisasi Slang Stopword Removal Stemming
Klasifikasi
TF-IDF Vectorizer Random Forest Decision Tree (Gini) Confusion Matrix

Cara Menggunakan
  1. Login menggunakan akun admin
  2. Impor dataset ulasan dari CSV
  3. Jalankan Preprocessing untuk pembersihan teks
  4. Latih model dengan tombol Latih Model
  5. Evaluasi akurasi di halaman Evaluasi
  6. Analisis sentimen teks baru di halaman Analisis
FAQ
Apakah data saya aman?

Ya, semua data disimpan di database lokal dan tidak dibagikan ke pihak ketiga manapun. Setiap akun memiliki data yang terpisah dan aman.

Bagaimana sentimen dihitung?

Aplikasi ini menggunakan algoritma Random Forest dengan fitur TF-IDF. Model dilatih dari dataset ulasan yang sudah dilabeli, kemudian digunakan untuk memprediksi sentimen teks baru.

Apakah ada batasan jumlah analisis?

Tidak ada batasan. Anda dapat melakukan analisis sebanyak yang Anda mau tanpa batasan apapun.
